For every command in notaro1997's ChatCommands plugin, there is a permission. The permission is (notaro.) then the name of the command, so for 
example if the command were /clear the permission would be notaro.clear, here is a list of every permission in the ChatCommands plugin, i think 
the command each permission corresponds with can be figured out, also if you want to know what the command does, you can try it out as op, op can 
override any permission, they're pretty self explanatory though. thanks for using my plugin. and have fun :) btw the sm command, you can spawn any 
mob by typing /sm then any mob for example /sm cow. also try out /sm crystal.
